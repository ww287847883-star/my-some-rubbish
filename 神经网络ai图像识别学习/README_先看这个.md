# CT 图像识别学习包

这个文件夹已经按“原始源码”和“学习整理版”分开，建议按下面顺序学习。

## 文件夹说明

1. `01_官方原始源码`
   - 保存 Keras 官方示例源码。
   - 不建议直接修改，方便以后和学习版对照。

2. `02_学习整理版`
   - 放你真正学习、运行、修改的文件。
   - `ct_3d_classification_learning.py` 是从官方源码复制出来的学习版。
   - `代码分段讲解.md` 用中文解释每一段代码在做什么。
   - `requirements.txt` 是需要安装的 Python 依赖。

3. `03_运行输出`
   - 建议以后把模型文件、训练曲线、截图、实验结果放这里。

## 推荐学习顺序

1. 先读 `02_学习整理版/代码分段讲解.md`
2. 再打开 `02_学习整理版/ct_3d_classification_learning.py`
3. 重点理解这几个部分：
   - CT 数据下载
   - `.nii` 医学影像读取
   - HU 值归一化
   - 3D CNN 模型结构
   - 训练和验证
   - 单个 CT 预测

## 注意

这个项目是学习用代码，不是临床诊断工具。医学影像模型需要严格的数据验证、医生标注、伦理审查和临床测试后才能用于真实医疗场景。
